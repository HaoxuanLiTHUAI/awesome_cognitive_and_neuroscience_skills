# Community Cases

> Featured usage cases curated from community contributions.

Community members share their real-world research experiences using skills in this repository via the [`share-case`](../../skills/share-case/) meta-skill. Maintainers select high-quality cases and archive them here.

## How Cases Get Here

1. A researcher uses a skill in their Claude Code session
2. They run the `share-case` meta-skill to share their experience
3. The case is posted to [GitHub Discussions > Show & Tell](https://github.com/HaoxuanLiTHUAI/awesome_cognitive_and_neuroscience_skills/discussions/categories/show-tell)
4. Maintainers select quality cases, add the `featured` label, and archive them here

## How to Share Your Case

In your Claude Code session after using any skill, say:

- "Share this case"
- "分享这个案例"

The `share-case` meta-skill will guide you through the process. You control exactly what gets shared.
